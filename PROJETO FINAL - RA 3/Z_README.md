# Projeto Final - RA 3

Este é um catálogo de livros com integração a uma API, onde o usuário pode visualizar, criar, editar e excluir registros(CRUD).

# Funcionalidades
-  Listagem de livros
-  Adicionar livros
-  Editar livros
-  Excluir livros
-  Alerta visual de sucesso

## Tecnologias utilizadas
- HTML5
- CSS3
- JavaScript (ES6+)
- Fetch API
- JSONPlaceholder API (mock)

## Como rodar o projeto

1. Clone este repositório:
   ```bash
   git clone https://github.com/RAUHANKM/PROJETO-FINAL-RA3
   
2. Abra o arquivo index.html no navegador ou IDE
  
3. Rode o live server

## Como usar
- Preencha o formulario com todas as informações necessarias
    TITULO
    NOME DO AUTOR
    SOBRE O LIVRO
    LINK DA CAPA EM FORMATO URL

- Clique em "Adicionar Livro" para incluir um novo registro.
- Clique no ícone de edição para atualizar dados.
- Clique no ícone de lixeira para excluir um livro.
