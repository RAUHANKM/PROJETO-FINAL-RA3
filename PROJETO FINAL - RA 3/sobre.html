Sobre nós

A Biblioteca Online é um projeto desenvolvido para facilitar o acesso a livros digitais e físicos, oferecendo um catálogo variado de obras para todos os públicos.

Buscamos aproximar a tecnologia do universo literário, tornando a experiência de pesquisa e empréstimo mais simples e rápida.

Este site foi desenvolvido por alunos do curso de Sistemas de Informação da PUCPR, como parte do projeto da disciplina de Web Development: Front-End.

Criadores:

Laura - Interface e estilização

Rauhan - Validações e UX Final

Isaque - CRUD e Lógica de Dados

Delrick - Estrutura, Navegação e Padrões de Código

Tecnologias utilizadas: HTML, CSS, JavaScript e Fetch API para integração com o banco de dados.

O projeto segue princípios de design responsivo, garantindo boa experiência em qualquer dispositivo.

Missão: Tornar a leitura acessível a todos.

Valores: Acesso à informação, sustentabilidade, inclusão e inovação.
